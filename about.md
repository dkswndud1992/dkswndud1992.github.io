# 🤖 로봇공학(Robotics) 소개

## "로봇(Robot)"이란 용어의 탄생

### 어원과 역사

**"로봇(Robot)"**이라는 단어는 **1920년** 체코슬로바키아의 극작가 **카렐 차페크(Karel Čapek)**가 쓴 희곡 **"R.U.R.(Rossum's Universal Robots)"**에서 처음 사용되었습니다.

- **어원**: 체코어 "robota(로보타)" - "강제 노동" 또는 "노역"을 의미
- **의미**: 인간을 대신해 일하는 인공적인 존재
- **역사적 의의**: 이 희곡은 로봇이 인간을 대체하고 결국 반란을 일으킨다는 내용으로, 현대 SF의 원형이 됨

### "로봇공학(Robotics)"의 창시자

**아이작 아시모프(Isaac Asimov)**는 1942년 단편 소설 "Runaround"에서 **로봇공학 3원칙**을 제시하며 "Robotics"라는 용어를 대중화했습니다:

1. 로봇은 인간에게 해를 끼쳐서는 안 되며, 위험에 처한 인간을 방관해서도 안 된다
2. 로봇은 제1원칙에 위배되지 않는 한 인간의 명령에 복종해야 한다
3. 로봇은 제1, 2원칙에 위배되지 않는 한 자신을 보호해야 한다

## � 글로벌 로봇 산업 동향

### 로봇 강국 Top 5

#### 1. 🇯🇵 일본
- **특징**: 산업용 로봇의 세계 1위 생산국
- **강점**: 
  - FANUC, Yaskawa, Kawasaki 등 세계적 로봇 제조사
  - 휴머노이드 로봇 기술 (ASIMO, Pepper)
  - 고령화 대응 로봇 (돌봄, 재활 로봇)
- **현황**: 세계 산업용 로봇의 약 45% 생산

#### 2. 🇰🇷 대한민국
- **특징**: 로봇 밀도 세계 1위 (제조업 근로자 10,000명당 로봇 1,000대)
- **강점**:
  - 반도체, 디스플레이 제조 자동화
  - 협동 로봇(Collaborative Robot) 기술
  - 서비스 로봇 및 AI 융합 로봇
- **주요 기업**: 삼성, LG, 현대, 두산, 한화 등

#### 3. 🇨🇳 중국
- **특징**: 세계 최대 산업용 로봇 시장 (전체의 50% 이상)
- **강점**:
  - 대규모 제조업 자동화 투자
  - 드론 및 물류 로봇 선도 (DJI 등)
  - AI + 로봇 융합 기술 발전
- **정책**: "Made in China 2025" 국가 전략

#### 4. 🇺🇸 미국
- **특징**: 로봇 연구개발 및 소프트웨어 강국
- **강점**:
  - Boston Dynamics (휴머노이드, 4족 보행 로봇)
  - 자율주행 및 AI 로봇 기술
  - 우주 탐사 로봇 (NASA Mars Rover)
  - 물류 자동화 (Amazon Robotics)
- **생태계**: 실리콘밸리 중심의 로봇 스타트업

#### 5. 🇩🇪 독일
- **특징**: 정밀 산업용 로봇 및 자동화 시스템 강국
- **강점**:
  - KUKA, ABB 등 세계적 로봇 제조사
  - Industry 4.0 (4차 산업혁명) 선도
  - 자동차 제조 자동화 기술

### 2024-2025 로봇 트렌드

- 🦾 **휴머노이드 로봇**: Tesla Optimus, Figure 01, Boston Dynamics Atlas
- 🚗 **자율주행**: Waymo, Tesla FSD, Cruise
- 🏭 **제조 자동화**: 협동 로봇(Cobot) 확산
- 📦 **물류 로봇**: Amazon, 쿠팡 물류센터 자동화
- 🏥 **의료 로봇**: 수술 로봇 (다빈치), 재활 로봇
- 🌾 **농업 로봇**: 자율 수확, 제초, 모니터링

## � 로봇공학 학습 접근법

### 1단계: 기초 이론 습득

#### 필수 수학
- **선형대수학**: 벡터, 행렬, 변환 (로봇 운동학의 핵심)
- **미적분학**: 속도, 가속도, 최적화
- **확률 및 통계**: 센서 융합, SLAM, 기계학습

#### 핵심 물리
- **고전 역학**: 뉴턴의 운동 법칙, 회전 운동
- **제어 이론**: PID 제어, 상태 공간 제어
- **전자공학**: 센서, 액추에이터, 회로 설계

### 2단계: 프로그래밍 실력 향상

```python
# 추천 언어
1. Python      # 프로토타이핑, AI/ML, ROS
2. C++         # 실시간 제어, 성능 최적화
3. MATLAB      # 시뮬레이션, 알고리즘 검증
```

### 3단계: 실습 중심 학습

#### 🛠️ 추천 실습 프로젝트
1. **라인 트레이싱 로봇** → 센서 + 제어 기초
2. **SLAM 로봇** → 자율 주행 핵심 기술
3. **매니퓰레이터** → 로봇 팔 제어 (역기구학)
4. **드론** → 비행 제어, 센서 융합

#### 🔧 추천 플랫폼
- **Arduino**: 초급 (간단한 제어)
- **Raspberry Pi**: 중급 (센서 처리, AI)
- **ROS (Robot Operating System)**: 고급 (전문가 수준)
- **Gazebo/PyBullet**: 시뮬레이션

### 4단계: 전문 분야 선택

```

├── 산업 자동화
├── 자율주행
├── 드론/항공 로봇
├── 의료 로봇
├── 휴머노이드
```

## 🎯 왜 로봇공학을 배워야 하는가?

### 1. 🌟 산업 전반의 패러다임 변화

> "로봇은 더 이상 SF가 아니라 일상이다"

- **제조업**: 무인 공장, 스마트 팩토리
- **물류**: 자율 배송, 창고 자동화
- **서비스**: 음식 배달, 청소, 안내 로봇
- **의료**: 원격 수술, 재활 치료
- **농업**: 자율 수확, 정밀 농업

### 2. 💼 높은 시장 수요

```
글로벌 로봇 시장 규모
2023년: $62.75 billion
2030년: $218.90 billion (예상)
연평균 성장률: 19.7%
```

**고임금 직업군**:

- AI/ML 로봇 전문가: $120,000 - $250,000/년

### 3. 🧠 융합 기술의 집합체

로봇공학은 다음을 모두 아우르는 종합 과학:
- 기계공학 (Mechanical Engineering)
- 전기/전자공학 (Electrical Engineering)
- 컴퓨터공학 (Computer Science)
- 인공지능 (Artificial Intelligence)
- 제어공학 (Control Engineering)

### 4. 🚀 미래 사회의 핵심 기술

- **고령화 사회**: 돌봄 로봇, 재활 로봇
- **노동력 부족**: 자동화, 무인화
- **위험 작업**: 재난 구조, 우주 탐사
- **생산성 향상**: 24시간 무중단 생산

## 🗂️ 로봇의 분류

### 1. 용도별 분류

#### 산업용 로봇 (Industrial Robots)
- **용접 로봇**: 자동차 제조
- **조립 로봇**: 전자제품 생산
- **도장 로봇**: 페인팅 자동화
- **핸들링 로봇**: 물류, 적재
- **협동 로봇(Cobot)**: 인간과 협업 가능

#### 서비스 로봇 (Service Robots)
- **가정용**: 청소 로봇 (Roomba), 잔디 깎기
- **의료용**: 수술 로봇 (다빈치), 재활 로봇
- **안내 로봇**: 호텔, 공항, 병원
- **배달 로봇**: 음식 배달, 택배
- **교육 로봇**: 코딩 교육, STEM

#### 특수 목적 로봇 (Special Purpose Robots)
- **군사 로봇**: 정찰, 폭발물 처리
- **우주 로봇**: 화성 탐사선, ISS 로봇 팔
- **수중 로봇**: 해저 탐사, 수중 작업
- **재난 구조 로봇**: 지진, 화재 구조

### 2. 구동 방식별 분류

#### 고정형 로봇
- **로봇 팔 (Manipulator)**: 6축, 7축 산업용 로봇
- **델타 로봇**: 고속 픽업 작업 (식품, 전자)
- **SCARA 로봇**: 조립, 검사 작업

#### 이동형 로봇 (Mobile Robots)
- **바퀴형 (Wheeled)**: 자율주행차, AGV
- **다리형 (Legged)**: 4족 보행 (Spot), 휴머노이드
- **비행형 (Aerial)**: 드론, 쿼드콥터
- **수중형 (Underwater)**: AUV, ROV

### 3. 지능 수준별 분류

#### Level 1: 사전 프로그래밍 로봇
- 정해진 경로와 동작만 수행
- 예: 전통적 산업용 로봇

#### Level 2: 센서 기반 로봇
- 센서로 환경 인식 후 반응
- 예: 청소 로봇, 라인 트레이싱 로봇

#### Level 3: 자율 로봇
- 스스로 판단하고 경로 계획
- 예: 자율주행차, SLAM 로봇

#### Level 4: AI 로봇
- 기계학습으로 학습하고 진화
- 예: 대화형 로봇, 적응형 제조 로봇

#### Level 5: AGI 로봇 (미래)
- 인간 수준의 범용 인공지능
- 현재 연구 단계

### 4. 구조별 분류

#### 직렬형 로봇 (Serial Robot)
```
Base → Link1 → Link2 → Link3 → End Effector
```
- 장점: 작업 공간 넓음
- 단점: 강성 낮음, 오차 누적
- 예: 대부분의 로봇 팔

#### 병렬형 로봇 (Parallel Robot)
```
     End Effector
    /    |    \
  L1    L2    L3
   \    |    /
      Base
```
- 장점: 고속, 고정밀
- 단점: 작업 공간 좁음
- 예: Stewart Platform, 델타 로봇

#### 하이브리드 로봇
- 직렬 + 병렬 구조 결합
- 예: 일부 산업용 로봇
